## Configure mac os

Disable inserting of special characters when long pressing keys:
`defaults write -g ApplePressAndHoldEnabled -bool false`

Quicker key repeats:
`defaults write -g InitialKeyRepeat -int 10`
`defaults write -g KeyRepeat -int 1`
